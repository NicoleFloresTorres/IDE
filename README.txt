# Analizador Léxico para Código C

Este proyecto es un analizador léxico para código en lenguaje C, que utiliza Python, Eel (para la interfaz web), Tkinter y expresiones regulares para analizar código, detectar tokens y errores léxicos. La interfaz se muestra en un navegador web y permite cargar, guardar archivos, y visualizar tokens y errores.

---

## Requisitos

- Python 3.7 o superior
- Módulos de Python:
  - eel
  - tkinter (normalmente viene con Python)
- Navegador web moderno (Chrome, Firefox, Edge, etc.)
